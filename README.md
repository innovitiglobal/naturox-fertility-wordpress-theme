## ğŸ› Troubleshooting

### Site doesn't look good / Colors and alignment are off

**Solution**: Make sure you've set the homepage template correctly:

1. Go to **Pages â†’ All Pages**
2. Edit your "Home" page
3. In the right sidebar under **Page Attributes**
4. Set **Template** to **"Home Page"**
5. Click **Update**
6. Go to **Settings â†’ Reading**
7. Ensure "Home" is selected as your homepage
8. **Clear your browser cache** (Ctrl+F5 or Cmd+Shift+R)

### Styles not loading

- Clear WordPress cache (if using a caching plugin)
- Clear browser cache
- Check if `style.css` exists in the theme folder
- Try deactivating and reactivating the theme

### Menu not showing

- The theme works without a custom menu
- If you want a custom menu, create one in **Appearance â†’ Menus**
- Assign it to "Primary Menu" location

### Logo not showing

- Upload logo in **Appearance â†’ Customize â†’ Site Identity**
